6 Temporal And-Or Graph

This chapter presents stochastic Temporal And-Or Graph (T-AOG) to represent the hierarchical compositions of events and the temporal relations between the sub-events.

6.1 A Simplified Atomic Action Model

6.1.1 Grounded Relations - the Alphabet

The T-AOG is defined on a set of unary and binary relations which can be directly detected from video. These relations are called the grounded relations:

  • A unary relation \(r(A)\) is a time varying property of the agent or object \(A\) in the scene. As Figure 6.1 shows, it could be agent poses, e.g. Stand(person1) and Bend(person2), and object states, e.g. Open(door) and Closed(door).
  • A binary relation \(r(A,B)\) is the spatial relation (e.g. Touch(person1.hand, phone)) between \(A\), \(B\) which could be agents, body parts (hands, feet), and objects. Figure 6.2 illustrates some typical binary relations.

Table 6.1 specifies the 24 unary and binary relations in the office scene. There are four types of relations: agent location (\(r_{01}\) ~ \(r_{13}\)), agent-environment interaction (\(r_{14}\) ~ \(r_{17}\)), agent pose (\(r_{18}\) ~ \(r_{21}\)) and environment event (\(r_{22}\) ~ \(r_{24}\)). Here we do not use the "Off" relation as shown in Figure 6.2 since we can infer the status of "Off" from the status of "On".
